# 📊 Análise de Dados com SQL - Dataset Olist

Este projeto foi desenvolvido como parte dos meus estudos em Análise de Dados, utilizando SQL para explorar e extrair insights de um dataset de e-commerce brasileiro (Olist).

Além dos estudos individuais, também participo de grupos de aprendizado como o **Data Girls**, o que tem contribuído muito para minha evolução prática na área.

---

## 🎯 Objetivo

Praticar consultas SQL aplicadas a um cenário real de negócio, explorando dados de pedidos, clientes, pagamentos e avaliações.

---

## 🔍 Análises Realizadas

Durante o projeto, foram desenvolvidas consultas para responder perguntas como:

- Quais pedidos foram entregues com atraso?
- Quais clientes realizaram mais de uma compra?
- Qual o volume de pedidos por estado?
- Qual a média de avaliação dos pedidos?
- Qual a receita total por estado?
- Como a avaliação varia de acordo com o status do pedido?
- Quais são as categorias de produtos mais vendidas?
- Análise completa por pedido (valor, frete, quantidade de itens e avaliação)

---

## 🛠️ Tecnologias Utilizadas

- SQL (SQLite Online)
- GitHub

---

## ⚠️ Observação

Os nomes das colunas foram mantidos conforme disponibilizados no ambiente de estudo (SQLite online / curso), podendo diferir do dataset original da Olist.

## 📈 Aprendizados

Este projeto foi importante para desenvolver e reforçar conceitos como:

- JOINs entre múltiplas tabelas  
- Funções de agregação (COUNT, SUM, AVG)  
- Agrupamentos com GROUP BY  
- Filtros com WHERE e HAVING  
- Organização e estruturação de consultas
